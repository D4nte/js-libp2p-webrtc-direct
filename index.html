Intro

Installable via npm install --save libp2p-webrtc-direct, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-webrtc-direct

You can then require() libp2p-webrtc-direct as normal:

const libp2PWebrtcDirect = require('libp2p-webrtc-direct')

In the Browser

Libp2p-webrtc-direct should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-webrtc-direct, including index.js and index.min.js, are included in the /dist folder. Libp2p-webrtc-direct can also be found on unpkg.com under

  • https://unpkg.com/libp2p-webrtc-direct/dist/index.min.js
  • https://unpkg.com/libp2p-webrtc-direct/dist/index.js
